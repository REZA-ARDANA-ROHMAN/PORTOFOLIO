Smart IoT Parking

Deskripsi Proyek

Smart IoT Parking adalah miniatur sistem parkir cerdas yang saya rancang untuk memantau jumlah slot parkir yang tersedia secara otomatis. Sistem menggunakan sensor ultrasonik, mikrokontroler ESP8266, dan server berbasis Laravel. Kendaraan yang masuk dan keluar terdeteksi secara real-time, lalu data ketersediaan parkir di-update ke server sehingga bisa dipantau melalui dashboard web.

Saya bertanggung jawab pada pengembangan hardware dan pemrograman mikrokontroler. Sistem ini membaca jarak kendaraan, mengendalikan palang parkir, dan menampilkan sisa slot parkir di LCD I2C, kemudian mengirim data ke server Laravel melalui Wi-Fi.

Tanggung Jawab Utama

  • Merancang dan merakit rangkaian elektronik sensor ultrasonik untuk mendeteksi kendaraan.
  • Memprogram ESP8266 NodeMCU menggunakan Arduino IDE untuk membaca jarak, mengendalikan servo palang parkir, dan menampilkan kondisi palang di LCD.
  • Membuat algoritma deteksi kendaraan masuk/keluar berdasarkan perbedaan jarak sensor.
  • Mengimplementasikan koneksi Wi-Fi dan protokol HTTP untuk mengirim data ke server.
  • Melakukan kalibrasi sensor agar sistem akurat mendeteksi kendaraan di berbagai kondisi.

Teknologi yang Digunakan

  • Hardware: ESP8266 NodeMCU, 2x sensor ultrasonik HC-SR04, servo motor, LCD I2C 16x2.
  • Bahasa Pemrograman: C++ (Arduino IDE).
  • Protokol Komunikasi: HTTP via Wi-Fi.
