EMBEDDED SYSTEM ESP32 SEBAGAI KONTROLER ROBOT TARI

Dalam proyek ini, saya merancang dan mengembangkan sistem embedded berbasis ESP32 yang berfungsi sebagai kontroler utama untuk robot tari.

Sistem ini diimplementasikan melalui pembuatan PCB custom 2-layer, yang saya desain secara mandiri untuk mendukung kebutuhan elektronik robot dalam menampilkan gerakan tari yang presisi dan sinkron dengan musik.

PCB ini dilengkapi dengan:

  • Modul ESP32 sebagai prosesor utama dengan konektivitas Wi-Fi dan Bluetooth, mendukung kontrol nirkabel dan sinkronisasi antar robot tari.
  • Driver motor dan sirkuit power management yang memberikan suplai daya stabil untuk performa servo motor yang halus.
  • Port antarmuka sensor untuk integrasi sensor gyroscope atau accelerometer.
  • Interface LCD untuk menampilkan status mode gerakan robot secara real-time.
  • Desain jalur PCB dengan routing optimal untuk minimalisasi noise dan efisiensi sinyal, serta tata letak mendukung kemudahan perawatan.
